# garasche.ch

## Setup

Requires `node`.

```sh
npm install
```

## Local development

```sh
npm run serve
```

## Tests

```sh
npm run test
```

## Build locally

(builds into `./dist`)

```sh
npm run build
```

## Deploy production

`npm run build` happens automatically on GitHub on `push` to `main` branch. See `.github/workflows/ci.yml`
